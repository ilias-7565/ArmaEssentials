
## Startup Parameters

(ignore GUI items)

Main 
* CPU: Intel, 4 cores / 8 processing Threads .
* RAM: 16 GB .
* GPU Dedicated Memory / VRAM: 4 GB .
```
-cpuCount=8
-exThreads=7
-enableHT
-maxMem=12032
-hugePages
-maxVRAM=4096
-noSplash
```

Additional and custom/{case specific}
```
-skipIntro
-noLogs
-noPause
-host
-loadMissionToMemory
<<"-noLand", "-world=empty" or "-world=Chernarus">>
-showScriptErrors
```
